# 🌦️ Weather Application

A simple and responsive **Weather Application** built using **HTML, CSS, and JavaScript** that provides real-time weather information for different locations.

---

# ✨ Features

- 🌍 Search weather by city name
- 🌡️ Displays temperature details
- ☁️ Weather condition information
- 💨 Wind speed display
- 💧 Humidity information
- 📱 Responsive user interface
- ⚡ Fast and lightweight application
